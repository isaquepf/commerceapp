# ApiGateway with Ocelot

This is a little bit but awesome example the microservices with api gateway.

## Configuration

Execute in root project: /commerce
```bash
dotnet build. 
```

## Usage


Execute each project:

/CatalogApi

/OrderApi

/GatewayApi

```bash
dotnet run
```
